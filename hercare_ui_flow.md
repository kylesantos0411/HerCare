# HerCare App – Full UI Flow & Navigation

Good choice. Option 1 has the best balance of usefulness and emotional value.

I’ll design the app as a clear mobile flow you can actually build. I’ll assume this is a phone app for your girlfriend, with a soft, clean, calm design.

App concept: **Nurse Shift Companion + Wellness Tracker**

Working name ideas:
- HerCare
- ShiftBloom
- CareNest
- Duty & Darling
- NurtureShift

I’ll use **HerCare** below as the sample name.

---

## 1. Core app structure

The app should feel simple, not clinical. A nurse already deals with enough stress, so the UI should be soft and low-friction.

Main modules:
- Home
- Shift
- Wellness
- Meals
- Notes
- You

Bottom navigation:
- Home
- Meals
- Wellness
- Notes
- You

Shift is accessed from Home.

---

## 2. App flow overview

### First-time user flow
Splash → Welcome → Personal Setup → Shift Setup → Wellness Goals → Notifications → Home

### Daily flow
Splash → Home Dashboard → features
